# File-System-Simulator
A group project made in a college course in 2023, written in c++. 
By using nodes and pointers a Unix-style filesystem is simulated.

This is a part of the project, I created the backend node management which is almost everything in this repository. other team members were tasked with creating the front-end ui/command structure, (That I don't have anymore), along with a method to save the file system via in an xml file between testruns.

Functionality includes:
- Adding and removing files and folders.
- Displaying the text path of a file or folder
- metadata of a file or folder
- permissions
  
If you'd like try it yourself, you can edit the main.cpp file accordingly using commands from Directory.cpp.
